# Techmarsal

A world-class, mobile-first blog platform built for the future of tech journalism.

## Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (with CSS Modules & Global Design System)
- **Icons**: Lucide React
- **Database**: PostgreSQL (Neon)
- **ORM**: Prisma

## Features
- Premium futuristic UI/UX
- SEO/GEO Optimized
- Custom CMS Dashboard
- AI-Powered Insights
- High Readability (Inter, Space Grotesk)

## Getting Started
```bash
npm install
npm run dev
```
